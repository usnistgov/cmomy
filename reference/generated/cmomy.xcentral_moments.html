cmomy.xcentral_moments#

cmomy.xcentral_moments(x, mom, *, w=None, axis=None, dim=None, mom_dims=None, broadcast=False)[source]#

Calculate central mom along axis.

Parameters:

  • x (DataArray or tuple of DataArray) – input data

  • mom (int or tuple of int) – Order or moments. If integer or length one tuple, then moments are for a single variable. If length 2 tuple, then comoments of two variables

  • w (array-like, optional) – if passed, should be able to broadcast to x. An exception is if w is a 1d array with len(w) == x.shape[axis]. In this case, w will be reshaped and broadcast against x

  • axis (int) – Axis to reduce along.

  • dim (hashable) – Dimension to reduce along.

  • dtype (dtype) – Optional dtype for output data.

  • mom_dims (hashable or tuple of hashable) – Name of moment dimensions. Defaults to ('xmom', 'umom')

  • broadcast (bool) – If True, and x=(x0, x1), then perform ‘smart’ broadcasting. In this case, if x1.ndim = 1 and len(x1) == x0.shape[axis], then broadcast x1 to x0.shape.

Returns:

output (DataArray) – array of shape shape + (mom,) or (mom,) + shape depending on value of last, where shape is the shape of x with axis removed, i.e., shape=x.shape[:axis] + x.shape[axis+1:]. Assuming last is True, output[…,0] is the total weight (or count), output[…,1] is the mean value of x, output[…,n] with n>1 is the nth central moment
